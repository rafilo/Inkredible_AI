## Guide to Inkreadible AI
Install first
```
npm install
```

Rename the ".env.example" file to .env and paste your api key from your OpenAI account into VITE_OPEN_AI_KEY variable

Start the frontend
```
npm run dev
```

Start the backend
```
cd server
node server.js
```
